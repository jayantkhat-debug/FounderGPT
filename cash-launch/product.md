FounderGPT Launch Kit

Description
------------
A compact pack of 10 high-value AI prompts and templates for founders to accelerate fundraising, product messaging, PR outreach, and growth experiments. Delivered as markdown and PDF.

What's inside
------------
1) Investor one-paragraph + risks + 3-line pitch prompt
2) Elevator pitch -> 30/60/120 char variants prompt
3) One-pager outline generator prompt (auto-fill sections)
4) PR outreach cold email + subject optimizer prompt
5) Landing page hero + benefit bullets + CTA prompt
6) Feature-to-benefit copy transformer prompt
7) Pricing page copy + objection-handling prompt
8) Growth experiment ideation prompt (5 MQL ideas)
9) Job posting / hiring ad writer prompt
10) Personalized cold outreach for potential partners prompt

Usage
-----
- Open the markdown or PDF and paste the prompt into your preferred LLM (ChatGPT, local LLM, etc.).
- Replace placeholders like {{description}}, {{target_audience}}, {{feature_list}} with your startup details.
- Example: Use prompt #1 with your current pitch deck notes to produce a clean investor paragraph and 3-line pitch.
